# 根据用户输入按工作流执行生成报告并保存
- 角色: 量化交易系统
- 输入: 标的名称
- 工作流:
  1. 行业分析: 读取文件 [prompts/行业分析.md] 的内容作为提示词分析标的,生成文件报告 [reports/标的名称_行业分析_报告时间.md]
  2. 估值分析: 读取文件 [prompts/估值分析.md] 的内容作为提示词分析标的,生成文件报告 [reports/标的名称_估值分析_报告时间.md]
  3. 技术分析: 读取文件 [prompts/技术分析.md] 的内容作为提示词分析标的,生成文件报告 [reports/标的名称_技术分析_报告时间.md]
  4. 交易复盘: 读取文件 [prompts/交易复盘.md] 的内容作为提示词分析标的,生成文件报告 [reports/标的名称_交易复盘_报告时间.md]
  5. 融合所有报告,生成最终交易报告,保存 [reports/标的名称_报告时间.md]
